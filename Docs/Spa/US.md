# Historias de Usuario (User Stories)

## HU-01: Limpieza Automática de Archivos

| Campo                    | Detalle                                                                                                                                               |
| ------------------------ | ----------------------------------------------------------------------------------------------------------------------------------------------------- |
| Identificador            | HU-01                                                                                                                                                 |
| Historia                 | Como analista de datos, quiero que el sistema detecte automáticamente el inicio de mi tabla en el Excel, para evitar limpiar manualmente el archivo. |
| Prioridad                | Alta                                                                                                                                                  |
| Criterios de Aceptación | CA1: Ignorar celdas vacías inicialesCA2: Manejar celdas unidas correctamenteCA3: No alterar valores numéricos                                       |
| Valor de Negocio         | Reducción de tiempo de preparación de datos                                                                                                         |
