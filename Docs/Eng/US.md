# User Stories

## US-01: Automatic File Cleaning

| Field               | Detail                                                                                                                                                        |
| ------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Identifier          | US-01                                                                                                                                                         |
| Story               | As a data analyst, I want the system to automatically detect where my table starts in the Excel file, so I do not need to manually clean it before uploading. |
| Priority            | High                                                                                                                                                          |
| Acceptance Criteria | AC1: Ignore initial empty cellsAC2: Handle merged cells correctlyAC3: Preserve numeric values without modification                                            |
| Business Value      | Reduces data preparation time                                                                                                                                 |

---

## US-02: Dynamic Dashboard with Filters

| Field               | Detail                                                                                                     |
| ------------------- | ---------------------------------------------------------------------------------------------------------- |
| Identifier          | US-02                                                                                                      |
| Story               | As a sales manager, I want to filter charts by categories to analyze specific business behaviors visually. |
| Priority            | High                                                                                                       |
| Acceptance Criteria | AC1: Charts update when filters are appliedAC2: User can show/hide chartsAC3: Responsive interface         |
| Business Value      | Improves decision-making capabilities                                                                      |

---

## US-03: Context-Aware Executive Summary

| Field               | Detail                                                                                                                     |
| ------------------- | -------------------------------------------------------------------------------------------------------------------------- |
| Identifier          | US-03                                                                                                                      |
| Story               | As a non-technical user, I want to select tags of interest so that the AI-generated summary focuses on what matters to me. |
| Priority            | High                                                                                                                       |
| Acceptance Criteria | AC1: At least 3 relevant insights includedAC2: User can provide additional inputAC3: Export available in PDF format        |
| Business Value      | Enables quick understanding of key insights                                                                                |

---

## US-04: Session Persistence

| Field               | Detail                                                                                                    |
| ------------------- | --------------------------------------------------------------------------------------------------------- |
| Identifier          | US-04                                                                                                     |
| Story               | As a recurring user, I want to save my dashboards in different chats so I can compare analyses over time. |
| Priority            | Medium                                                                                                    |
| Acceptance Criteria | AC1: Anonymous first use allowedAC2: Registration required for multiple sessionsAC3: Full CRUD for chats  |
| Business Value      | Enables continuity and comparison of analyses                                                             |
